Percentage Based Grid

Columns widths are based on percentage, handy when widths are not uniform/consistent. Any combination works provided width never exceeds 100% (i.e. 60/40, 70/30, 80/20 etc..)

Online tools used: http://www.responsivegridsystem.com/calculator/

All percentage based columns have spacing by default (1.6%)

NOTE! additional calasses used for spacing, border and background colour. .in_tight .out_tight .out_only_bottom .outline .fill
