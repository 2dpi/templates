#### Button Click ####
```javascript
onClick="ga('send', 'event', {
    eventCategory: 'Category',
    eventAction:  'Action',
    eventLabel: 'Label'
});"
```

- Category - ie. button
- Action - ie. clicked
- Label - ie. contact form
