CSS Grid - Flush Fitting

Override columns widths based on viewport size using media queries to achive required results.

Popular devices to target are iPhone, iPad, Samsung Galaxy Sx etc refer to http://viewportsizes.com for reference

{% highlight css linenos %} {% raw %} @media screen and (max-width: 320px) { /* smart phone portrait */ body { color: #C00; } } @media screen and (max-width: 568px) { /* smart phone landscape (iPhone)*/ body { color: #C00; } } @media screen and (max-width: 640px) { /* smart phone landscape (samsung S*)*/ body { color: #C00; } } @media screen and (max-width: 767px) { /* small tablet portrait */ body { color: #C00; } } {% endraw %} {% endhighlight %}
