CSS Grid - Responsive

Override columns widths based on viewport size using media queries to achive required results.

To achive good results when the user uses the browser zoom function it's advisable to implement the use of em's instead of px's

http://pxtoem.com/ is a useful tool to help with the conversion. e.g. Browser defaut font size is 16px so to set a breakpoint at 300px = 18.750em

Popular devices to target are iPhone, iPad, Samsung Galaxy Sx etc refer to http://viewportsizes.com for reference

{% highlight css linenos %} {% raw %} @media screen and (max-width: 320px) { /* smart phone portrait */ .container_article { width: 100%; } } @media screen and (max-width: 480px) { /* smart phone portrait */ .container_article { width: 100%; } } @media screen and (max-width: 568px) { /* smart phone landscape (iPhone)*/ .container_article { width: 100%; } } @media screen and (max-width: 640px) { /* smart phone landscape (samsung S*)*/ .container_article { width: 100%; } } @media screen and (max-width: 768px) { /* small tablet portrait */ .container_article { width: 100%; } } {% endraw %} {% endhighlight %}
