CSS Grid - Responsive

Override columns widths based on viewport size using media queries to achive required results.

Popular devices to target are iPhone, iPad, Samsung Galaxy Sx etc refer to http://viewportsizes.com for reference

{% highlight css linenos %} {% raw %} @media screen and (max-width: 320px) { /* smart phone portrait */ .container_article { width: 100%; } } @media screen and (max-width: 568px) { /* smart phone landscape (iPhone)*/ .container_article { width: 100%; } } @media screen and (max-width: 640px) { /* smart phone landscape (samsung S*)*/ .container_article { width: 100%; } } @media screen and (max-width: 767px) { /* small tablet portrait */ .container_article { width: 100%; } } {% endraw %} {% endhighlight %}
