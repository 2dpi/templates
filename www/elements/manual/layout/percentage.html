CSS Grid - Flush Fitting

Columns widths are based on percentage, handy when widths vary from standard grid. Any combination works provided width never exceeds 100% (i.e. 60/40, 70/30, 80/20 etc..)

Online tools used: http://www.responsivegridsystem.com/calculator/

All percentage based columns have spacing by default (1.6%)
