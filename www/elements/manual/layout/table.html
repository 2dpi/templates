CSS Table

Cell widths are based on percentage, handy when widths are not uniform/consistent. Any combination works provided width never exceeds 100% (i.e. 60/40, 70/30, 80/20 etc..)

Mimicking HTML Table behaviour forces equal heights on targeted elements and solves a common styling issue.

"Cells" will also have equal width unless otherwise specified.

NOTE! additional calasses used for spacing, border and background colour. .in_tight .out_tight .out_only_bottom .outline .fill
