The aim of the w3 framework is to provides basic building blocks to aid in rapidly deploying a website. Further objective are:

  • modular
  • accessible
  • responsive
  • cross browser consistancy

Modular

For development purposes code is divided into:

  • layout
  • navigation
  • media
  • elements
  • scripts
  • style

The reasoning would be that layout, navigation, media, elements and scripts would form a foundation that can be expanded, updated and maintained without affecting the style/theme of the site.

Accessible

This is a work in progress and will get more attention once the foundation settles.
